## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/DanielMandira/Calculadora-IMC.git
```

Entre no diretório do projeto
```
  Calculadora IMC\bin\Debug\net6.0-windows
```
Execute o programa
```
  calculadora IMC.exe
```
